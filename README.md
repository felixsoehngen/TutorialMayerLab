# Coffee, Git, and Snakemake Tutorial

A tiny chemistry-themed onboarding project for practicing terminal navigation, Git/GitHub, VS Code, conda, Jupyter, pandas, seaborn, and a minimal Snakemake workflow.

## Quick start
1. `conda env create -f environment.yml`
2. `conda activate chem-coffee-tutorial`
3. `jupyter lab`
4. Open `notebooks/01_coffee_git_snakemake_tutorial.ipynb`
5. Later, run `snakemake -j1`
